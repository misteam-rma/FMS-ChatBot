================================================================================
                          ❓ HOW AUTO-SETUP WORKS
================================================================================

When your app starts (first time):

1. FastAPI initializes
2. SQLAlchemy creates database tables
3. auto_setup.py runs automatically:
   - Checks if company exists
   - If NOT: Creates company record
   - If NOT: Creates database connection pointing to your GOOGLE_SHEET_ID
   - Prints success message in logs
4. App is ready to use

On subsequent restarts:
   - Detects company already exists
   - Skips setup
   - Continues normally

✅ NO MANUAL STEPS - Everything automatic!

================================================================================
                          🔑 REQUIRED CREDENTIALS
================================================================================

1. GOOGLE_SERVICE_ACCOUNT_JSON
   - Get from: Google Cloud Console → Service Accounts
   - Download JSON file
   - Copy entire content as single line (no line breaks)
   - Paste in .env

2. OPENAI_API_KEY
   - Get from: https://platform.openai.com/account/api-keys
   - Create new API key
   - Paste in .env

3. GOOGLE_SHEET_ID
   - From URL: https://docs.google.com/spreadsheets/d/{SHEET_ID}/edit
   - Copy the ID part
   - Paste in .env

4. JWT_SECRET_KEY
   - Generate random string:
     python -c "import secrets; print(secrets.token_hex(32))"
   - Paste in .env

================================================================================
                          🆘 TROUBLESHOOTING
================================================================================

❌ "Module not found: uvicorn"
   → pip install -r requirements.txt

❌ "Unable to load PEM file"
   → GOOGLE_SERVICE_ACCOUNT_JSON is invalid JSON
   → Make sure it's one line with no line breaks

❌ "No GOOGLE_SHEET_ID found"
   → Add GOOGLE_SHEET_ID to .env
   → Restart application

❌ Auto-setup failed in logs
   → Check GOOGLE_SHEET_ID is correct
   → Check GOOGLE_SERVICE_ACCOUNT_JSON is valid
   → Restart app once fixed

❌ Login says "mobile number not found"
   → Verify phone number is in Google Sheet
   → Verify phone column is named "Phone Number"
   → Check employee data is in "Employee Master Data" sheet

❌ Chat not responding
   → Check OPENAI_API_KEY is valid
   → Check OpenAI account has credits
   → Check internet connection
