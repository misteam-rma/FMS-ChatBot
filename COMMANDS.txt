================================================================================
                            TROUBLESHOOTING
================================================================================

❌ "Unable to load PEM file"
   → Check GOOGLE_SERVICE_ACCOUNT_JSON is valid JSON (one line, no line breaks)

❌ "No GOOGLE_SHEET_ID found"
   → Add GOOGLE_SHEET_ID to .env and restart

❌ "Module not found: uvicorn"
   → Run: pip install -r requirements.txt

❌ "Connection refused to localhost:8000"
   → Check backend is running and port 8000 is not blocked

❌ Database not auto-setup
   → Check logs, restart app with correct GOOGLE_SHEET_ID
